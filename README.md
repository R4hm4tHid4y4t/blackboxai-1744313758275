# Fanya Publishing - Penerbit Buku Indonesia

## Project Overview
Fanya Publishing is an online platform for book publishing in Indonesia, providing writers an easy way to publish their works. The site features various services including individual book publishing, collaborative book projects, and an extensive catalog of available books for sale. Users can navigate through a user-friendly interface and access various sections including catalogs, publishing services, and user accounts.

## Installation
To run the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate into the project directory:
   ```bash
   cd fanya-publishing
   ```

3. Open the `index.html` file in a web browser to view the project.

4. To enable local development, an HTTP server can be used (e.g., using Python):
   ```bash
   python -m http.server
   ```

## Usage
- **Home Page**: View promotional content and links to various sections of the site.
- **Catalog**: Browse the list of available books, filter by categories, and view detailed book information.
- **Publishing Services**: Learn about different publishing options offered by Fanya Publishing.
- **User Account**: Users can log in to view their profiles, orders, and wishlist.
- **Checkout Process**: Add books to the cart and proceed to checkout to make purchases.

## Features
- **User Registration and Login**: Users can create accounts and log in to manage their profiles.
- **Book Catalog**: Users can browse books, view details, and filter by categories.
- **Cart System**: Users can add books to their cart and proceed to checkout.
- **Responsive Design**: The website is designed for optimal viewing on various devices, from mobile phones to desktops.

## Dependencies
The project uses the following dependencies, which are included in the `package.json` file:
- **Tailwind CSS**: A utility-first CSS framework to style the components.
- **Font Awesome**: For integrating vector icons in the interface.
- **Google Fonts**: For beautiful typography through the Inter font.

## Project Structure
The project directory contains the following files:

- **index.html**: The main entry point for the user interface.
- **catalog.html**: Displays the book catalog, including filters and individual book details.
- **book-detail.html**: Provides detailed information about a selected book.
- **publishing-services.html**: Outlines various publishing services offered.
- **login.html**: The user login interface.
- **register.html**: The user registration interface to create a new account.
- **dashboard.html**: User dashboard for managing profiles and viewing past orders.
- **cart.html**: Displays items in the user's shopping cart with a summary.
- **checkout.html**: Handles the checkout process for finalizing purchases.
- **about.html**: Contains information about the company.
- **contact.html**: A form for users to contact the publishing house for inquiries.

The structure is organized to provide an easy navigation flow and maintainability.

## Conclusion
Fanya Publishing aims to simplify the book publishing process for authors while offering readers an easy platform to discover new books. The project is an excellent demonstration of HTML, CSS, and JavaScript working together to deliver a functional and appealing web application. For further improvements, consider expanding the backend functionalities for a complete publishing management system.
